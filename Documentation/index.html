Overview

Extend your professional bio to the next level with “CVitae” - a clean, minimal, parallax resume/portfolio WordPress theme . In CVitae, the Materialized CSS Effects is implemented correctly to attract the viewers to the fullest. Check the creative professional intros (14+) and get your online resume done at a glimpse.

Features


  • Free fully layered editable resume in PSD / AI / MsWord
  • Demo data importer
  • Fully covered backend, easy to use
  • No coding skill required! Our backend has it all!
  • 14 different intros to choose from
  • Clean and validated code
  • Materialized environment
  • 100% Responsive
  • Adaptive navigation menu
  • Clean & minimal design
  • Section based layout
  • Super fast loading
  • Pure CSS made intros
  • Blog posts feature included
  • Retina ready
  • Easy color customization
  • Sticky navigation bar
  • Cross Browser Compatible – IE9+, Chrome, Firefox, Safari, Opera
